# <FiGlobe className="inline-block mr-2" /> Introduction

**Kenat** (Amharic: ቀናት) is a multi-language toolkit for working with the **Ethiopian calendar system** — including support for conversion, date arithmetic, calendar grids, Geez numerals, and more.

Whether you're building a web app, desktop utility, backend scheduler, or mobile experience, **Kenat gives you consistent tools** across languages.

---

## <FiTool className="inline-block mr-2" /> Key Features

- <FiRefreshCw className="inline-block mr-1" /> Convert between **Ethiopian ↔ Gregorian** dates
- <FiCalendar className="inline-block mr-1" /> Full **13-month** Ethiopian calendar support (with Pagume)
- <FaCalculator className="inline-block mr-1" /> **Add/subtract** days, months, and years
- <FaSortNumericDown className="inline-block mr-1" /> Format numbers and dates in **Geez numerals**
- <FiClock className="inline-block mr-1" /> Work with traditional **Ethiopian time**
- <FiCalendar className="inline-block mr-1" /> Generate **calendar grids**
- <FaLandmark className="inline-block mr-1" /> Detect and label **holidays**
- <FiGlobe className="inline-block mr-1" /> Localized output in Amharic, English, and more

    ```dart
    }
    ```

    ```python
    print(date.format_in_geez_amharic())
    # ግንቦት ፳፭ ፳፻፲፯
    ```
