## Install via npm
```bash
npm install -g kenat-cli
```

Or add to your project:
```bash
npm install kenat-cli
```

## Requirements
- Node.js v16 or newer

## Usage
After installation, run:
```bash
kenat
```
This starts interactive mode. You can also run specific commands directly:
```bash
kenat today
kenat convert 2016/09/15 --to gregorian
kenat holiday --this-month
kenat calendar --year 2017 --month 1
kenat bahirehasab 2017
```

Type `kenat help` for usage instructions.
