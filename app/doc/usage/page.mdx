## <FaCalculator className="inline-block mr-2" /> Date Arithmetic

<Tabs
  items={[
    <div key="javascript" style={{ display: 'flex', alignItems: 'center', gap: '0.4rem' }}><SiJavascript size={16} /> JavaScript</div>,
    <div key="dart" style={{ display: 'flex', alignItems: 'center', gap: '0.4rem' }}><SiDart size={16} /> Dart</div>,
    <div key="python" style={{ display: 'flex', alignItems: 'center', gap: '0.4rem' }}><SiPython size={16} /> Python</div>,
    <div key="php" style={{ display: 'flex', alignItems: 'center', gap: '0.4rem' }}><SiPhp size={16} /> PHP</div>
  ]}
>
  <Tabs.Tab>
    ```javascript
    const k = new Kenat('2015/6/10');

    // These methods mutate the instance and are chainable
    k.addDays(5).addMonths(-1).addYears(2);

    console.log(k.getEthiopian());
    // → { year: 2017, month: 5, day: 15 }
    ```
  </Tabs.Tab>
  <Tabs.Tab>
    ```dart
    var k = Kenat('2015/6/10');

    // These methods are chainable, but (unlike JS) each one returns a new,
    // immutable Kenat instance rather than mutating in place — reassign.
    k = k.addDays(5).addMonths(-1).addYears(2);

    print(k.getEthiopian());
    // → {year: 2017, month: 5, day: 15}
    ```
  </Tabs.Tab>
  <Tabs.Tab>
    ```python
    from kenat import Kenat
    from kenat.day_arithmetic import add_days, add_months, add_years

    k = Kenat('2015/6/10')

    # The functions return a new Kenat instance
    k = add_days(k, 5)
    k = add_months(k, -1)
    k = add_years(k, 2)

    print(k.get_ethiopian())
    # → {'year': 2017, 'month': 5, 'day': 15}
    ```
  </Tabs.Tab>
  <Tabs.Tab>
    ```php
    use Kenat\Kenat;

    $k = new Kenat('2015/6/10');

    // Kenat is immutable in PHP too: these methods return a NEW instance,
    // but they still chain.
    $k = $k->addDays(5)->addMonths(-1)->addYears(2);

    $et = $k->getEthiopian();
    echo "{$et->year}-{$et->month}-{$et->day}";
    // → 2017-5-15
    ```
  </Tabs.Tab>
</Tabs>

## <FiCalendar className="inline-block mr-2" /> Generate a Month Grid

<Tabs
  items={[
    <div key="javascript" style={{ display: 'flex', alignItems: 'center', gap: '0.4rem' }}><SiJavascript size={16} /> JavaScript</div>,
    <div key="dart" style={{ display: 'flex', alignItems: 'center', gap: '0.4rem' }}><SiDart size={16} /> Dart</div>,
    <div key="python" style={{ display: 'flex', alignItems: 'center', gap: '0.4rem' }}><SiPython size={16} /> Python</div>,
    <div key="php" style={{ display: 'flex', alignItems: 'center', gap: '0.4rem' }}><SiPhp size={16} /> PHP</div>
  ]}
>
  <Tabs.Tab>
    ```javascript
    import { MonthGrid } from 'kenat';

    const grid = new MonthGrid(2017, 9, { useGeez: true });

    console.log(grid.headers);
    // → [ "እሑድ", "ሰኞ", "ማክሰኞ", ... ]

    console.log(grid.days[0]);
    // → { ethiopian: { day: 1 }, gregorian: { day: 9 }, holiday: null }
    ```
  </Tabs.Tab>
  <Tabs.Tab>
    ```dart
    import 'package:kenat/kenat.dart';

    // createMonthGrid() returns a Map<String, dynamic> directly
    final grid = createMonthGrid({ 'year': 2017, 'month': 9, 'useGeez': true });

    print(grid['headers']);
    // → [ሰኞ, ማክሰኞ, ረቡዕ, ሐሙስ, ዓርብ, ቅዳሜ, እሑድ]

    // grid['days'] is a flat, weekStart-padded list — leading entries can be
    // null, so find the first real day rather than assuming index 0:
    final days = grid['days'] as List;
    final firstDay = days.firstWhere((d) => d != null);
    print(firstDay);
    // → {ethiopian: {year: ፳፻፲፯, month: ግንቦት, day: ፩}, gregorian: {year: 2025, month: 5, day: 9}, weekday: 5, weekdayName: ዓርብ, isToday: false, holidays: []}
    ```
  </Tabs.Tab>
  <Tabs.Tab>
    ```python
    from kenat.month_grid import MonthGrid

    # The create method returns the grid directly
    grid = MonthGrid.create(year=2017, month=9, use_geez=True)

    print(grid['headers'])
    # → [ 'እሑድ', 'ሰኞ', 'ማክሰኞ', ... ]

    # Accessing a day in the grid
    # The grid is a list of weeks, and weeks are lists of days
    first_day_of_first_week = grid['days'][0][0]
    print(first_day_of_first_week)
    ```
  </Tabs.Tab>
  <Tabs.Tab>
    ```php
    use Kenat\MonthGrid;

    // MonthGrid::create() returns an array directly — same shape as Dart's
    $grid = MonthGrid::create(['year' => 2017, 'month' => 9, 'useGeez' => true]);

    print_r($grid['headers']);
    // → [እሑድ, ሰኞ, ማክሰኞ, ...]

    // $grid['days'] is a flat, weekStart-padded array — leading entries can
    // be null, so find the first real day rather than assuming index 0:
    $firstDay = array_values(array_filter($grid['days'], fn ($d) => $d !== null))[0];
    print_r($firstDay);
    // → ['ethiopian' => [...], 'gregorian' => [...], 'weekday' => ..., 'holidays' => [...]]
    ```
  </Tabs.Tab>
</Tabs>

## <FaMosque className="inline-block mr-2" /> Built-in Holidays

<Tabs
  items={[
    <div key="javascript" style={{ display: 'flex', alignItems: 'center', gap: '0.4rem' }}><SiJavascript size={16} /> JavaScript</div>,
    <div key="dart" style={{ display: 'flex', alignItems: 'center', gap: '0.4rem' }}><SiDart size={16} /> Dart</div>,

    <div key="php" style={{ display: 'flex', alignItems: 'center', gap: '0.4rem' }}><SiPhp size={16} /> PHP</div>
  ]}
>
  <Tabs.Tab>
    ```javascript
    import { getHoliday, getHolidaysForYear, getHolidaysInMonth, HolidayTags } from 'kenat';

    // Get a holiday by key and year
    const enkutatash = getHoliday('enkutatash', 2017);
    console.log(enkutatash);
    // → { key: 'enkutatash', tags: [...], name: 'አዲስ ዓመት', description: 'Ethiopian New Year', ethiopian: { year: 2017, month: 1, day: 1 } }

    // Get all public holidays for a year
    const publicHolidays = getHolidaysForYear(2017, { filter: HolidayTags.PUBLIC });
    console.log(publicHolidays);

    // Get holidays for a specific month
    const monthHolidays = getHolidaysInMonth(2017, 1, { filter: HolidayTags.PUBLIC });
    console.log(monthHolidays);

    // Use language option
    const meskelEn = getHoliday('meskel', 2017, { lang: 'english' });
    console.log(meskelEn.name); // 'Meskel'
    ```

    ### <FiAlertTriangle className="inline-block mr-2" /> Error Handling
    If you provide an invalid key or date, the functions return null or throw an error.
  </Tabs.Tab>
  <Tabs.Tab>
    ```dart
    import 'package:kenat/kenat.dart';

    void main() {
      // Get a holiday by key and year
      final enkutatash = getHoliday('enkutatash', 2017);
      print(enkutatash);
      // → {key: enkutatash, tags: [public, cultural], movable: false, name: እንቁጣጣሽ, description: ..., ethiopian: {year: 2017, month: 1, day: 1}}

      // Get all public holidays for a year
      final publicHolidays = getHolidaysForYear(2017, { 'filter': HolidayTags.public });
      print(publicHolidays.length); // 15

      // Get holidays for a specific month
      final monthHolidays = getHolidaysInMonth(2017, 1, { 'filter': HolidayTags.public });
      print(monthHolidays.length); // 3

      // Use language option
      final meskelEn = getHoliday('meskel', 2017, { 'lang': 'english' });
      print(meskelEn?['name']); // 'Finding of the True Cross (Meskel)'
    }
    ```

    ### <FiAlertTriangle className="inline-block mr-2" /> Error Handling
    `getHoliday` returns `null` for an unknown key; functions that require a
    valid Ethiopian year (like `getHolidaysInMonth`) throw a typed error
    (e.g. `InvalidInputTypeError`) for out-of-range input.
  </Tabs.Tab>
  <Tabs.Tab>
    ```python
    from kenat.holidays import get_holiday

    # Get holiday by its key and year
    enkutatash = get_holiday('enkutatash', 2017)
    print(enkutatash)
    # → {'key': 'enkutatash', 'tags': ['national', 'christian'], 'movable': False, 'name': 'አዲስ ዓመት', 'description': 'Ethiopian New Year', 'ethiopian': {'year': 2017, 'month': 1, 'day': 1}}
    ```
  </Tabs.Tab>
  <Tabs.Tab>
    ```php
    use Kenat\Holidays;
    use Kenat\HolidayTags;

    // Get a holiday by key and year — returns a typed Holiday value object, or null
    $enkutatash = Holidays::getHoliday('enkutatash', 2017);
    echo "{$enkutatash->name}: {$enkutatash->ethiopian->month}/{$enkutatash->ethiopian->day}";
    // → አዲስ ዓመት: 1/1

    // Get all public holidays for a year
    $publicHolidays = Holidays::getHolidaysForYear(2017, ['filter' => HolidayTags::PUBLIC]);
    echo count($publicHolidays); // 15

    // Get holidays for a specific month
    $monthHolidays = Holidays::getHolidaysInMonth(2017, 1, ['filter' => HolidayTags::PUBLIC]);
    echo count($monthHolidays); // 3

    // Use the lang option
    $meskelEn = Holidays::getHoliday('meskel', 2017, ['lang' => 'english']);
    echo $meskelEn->name; // 'Meskel'
    ```

    ### <FiAlertTriangle className="inline-block mr-2" /> Error Handling
    `getHoliday` returns `null` for an unknown key; functions that require a
    valid Ethiopian date throw a typed exception (e.g. `InvalidInputTypeException`).
  </Tabs.Tab>

# <FaBookOpen className="inline-block mr-2" /> Quick Reference: Named Exports

Kenat exports many useful functions and constants:

- `Kenat` (default)
- `MonthGrid`
- `Time`
- `toEC`, `toGC`, `toGeez`, `toArabic`
- `getHolidaysForYear`, `getHolidaysInMonth`, `getHoliday`
- `HolidayTags`, `monthNames`
- `getBahireHasab`, `getFastingPeriod`
- Formatting: `formatStandard`, `formatWithWeekday`, `formatWithTime`, `formatInGeezAmharic`, `formatShort`, `toISODateString`

**PHP equivalent:** same functionality, organized as static methods on
namespaced classes instead of top-level functions — `Kenat`, `MonthGrid`,
`Time`, `Conversions::toEC/toGC`, `GeezConverter::toGeez/toArabic`,
`Holidays::getHolidaysForYear/getHolidaysInMonth/getHoliday`, `HolidayTags`,
`BahireHasab::calculate/getMovableHoliday`, `Fasting::getFastingPeriod`, and
`Formatting::formatStandard/formatWithWeekday/formatWithTime/formatInGeezAmharic/formatShort/toISODateString`.
