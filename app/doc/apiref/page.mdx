# ‚öôÔ∏è Kenat Class

The `Kenat` class provides a high-level API for working with Ethiopian calendar dates. You can create instances, format dates, manipulate them, and print calendars.

---

## üèó Constructor

`new Kenat(dateString?)`

Creates a new instance of an Ethiopian date.

- If no argument is passed, it uses the current Gregorian system date.
- If a string like `'2015/6/10'` is passed, it's parsed as an Ethiopian date.

## üìÜ Date Access Methods

| Method            | Returns                          | Description                          |
|-------------------|-----------------------------------|--------------------------------------|
| `getEthiopian()` / `get_ethiopian()`  | `{ year, month, day }`            | Current Ethiopian date               |
| `getGregorian()` / `get_gregorian()`  | `{ year, month, day }`            | Equivalent Gregorian date            |
| `toString()` / `__str__()`      | `"Ethiopian: yyyy-mm-dd"`         | String format                        |
| `format(lang?)`   | `"·àò·àµ·ä®·à®·àù 5 2016"` or similar       | Human-readable string                |
| `formatInGeezAmharic()` / `format_in_geez_amharic()` | `"·àò·àµ·ä®·à®·àù ·ç≠ ·ç≥·çª·ç≤·çÆ"`         | Geez-styled output                   |

    <Tabs.Tab>

        console.log(k.getEthiopian()); // { year: 2016, month: 1, day: 5 }

## üïí Time Handling

| Method               | Description                                        |
|----------------------|----------------------------------------------------|
| `getCurrentTime()` / `get_current_time()`   | Gets current time in Ethiopian format              |
| `setTime(hour, min, period)` / `set_time(hour, min, period)` | Sets Ethiopian time manually (e.g., 3:30 night)  |

## ‚ûï Date Arithmetic

| Method              | Description                              |
|---------------------|------------------------------------------|
| `addDays(n)` / `add_days(n)`        | Move n days forward/backward             |
| `addMonths(n)` / `add_months(n)`      | Adjust by n Ethiopian months             |
| `addYears(n)` / `add_years(n)`       | Move by n Ethiopian years                |

<Tabs
  items={[

## üìè Compare Dates

| Method                | Description      |
| --------------------- | ---------------- |
| `diffInDays(other)` / `diff_in_days(other)`   | Day difference   |
| `diffInMonths(other)` / `diff_in_months(other)` | Month difference |
| `diffInYears(other)` / `diff_in_years(other)`  | Year difference  |
