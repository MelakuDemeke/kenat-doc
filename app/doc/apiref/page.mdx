# ‚öôÔ∏è Kenat Class

The `Kenat` class provides a high-level API for working with Ethiopian calendar dates. You can create instances, format dates, manipulate them, and print calendars.

---

## üèó Constructor

`new Kenat(dateStringOrDict?)`

Creates a new instance of an Ethiopian date.

- If no argument is passed, it uses the current Gregorian system date.
- If a string like `'2015/6/10'` is passed, it's parsed as an Ethiopian date.
- It can also be initialized with a dictionary: `{'year': 2015, 'month': 6, 'day': 10}`.

## üìÜ Date Access & Formatting

Multiple methods are available to get the date in different formats.

| Method                               | Returns                               | Description                                      |
|--------------------------------------|---------------------------------------|--------------------------------------------------|
| `get_ethiopian()`                    | `{ 'year', 'month', 'day' }`          | Get the Ethiopian date as a dictionary.          |
| `get_gregorian()`                    | `{ 'year', 'month', 'day' }`          | Get the equivalent Gregorian date.               |
| `__str__()`                          | `"YYYY/MM/DD"`                        | A simple string representation.                  |
| `format_standard(lang='amharic')`    | `"·àò·àµ·ä®·à®·àù 5 2016"` or similar        | Human-readable string with language options.     |
| `format_in_geez_amharic()`           | `"·àò·àµ·ä®·à®·àù ·ç≠ ·ç≥·çª·ç≤·çÆ"`                  | Geez-styled output.                              |
| `format_with_weekday(lang, use_geez)`| `"·àõ·ä≠·à∞·äû, ·àò·àµ·ä®·à®·àù 1 2016"`            | Includes the day of the week.                    |
| `format_with_time(time_obj, lang)`   | `"·àò·àµ·ä®·à®·àù 10 2016 08:30 ·å†·ãã·âµ"`        | Combines date and a `Time` object.               |
| `to_iso_date_string(time_obj)`       | `"YYYY-MM-DDTHH:mm"`                  | ISO 8601-like format.                            |

## üïí Time Handling

The `Time` class handles time calculations, conversions, and formatting separately from the date.

| Method/Class             | Description                                                |
|--------------------------|------------------------------------------------------------|
| `Time(hour, min, period)`| Creates a time object (e.g., 3:30, 'night').               |
| `Time.from_gregorian(h, m)`| Creates an Ethiopian `Time` object from Gregorian 24h time.|
| `time.to_gregorian()`    | Converts Ethiopian time back to a 24h dictionary.          |
| `time.format()`          | Formats the time into a string (e.g., "·ç´:·ç¥ ·àå·àä·âµ").         |
| `time.add(duration)`     | Adds hours/minutes to the time.                            |
| `time.diff(other_time)`  | Calculates the difference between two `Time` objects.      |

## ‚ûï Date Arithmetic

These functions are available in the `kenat.day_arithmetic` module and take a `Kenat` object as the first argument.

| Function            | Description                              |

# üß© Advanced Features & Utilities

## Kenat Static Methods

- `Kenat.now()` ‚Äî Returns a Kenat instance for the current date/time.
- `Kenat.getMonthCalendar(year, month, options)` ‚Äî Returns a calendar grid for a given month.
- `Kenat.getYearCalendar(year, options)` ‚Äî Returns a full year calendar.
- `Kenat.generateDateRange(startDate, endDate)` ‚Äî Returns an array of Kenat dates for a range.

## MonthGrid Usage

You can generate a month grid with holidays and weekday headers:

```js
import { MonthGrid } from 'kenat';
const grid = new MonthGrid({ year: 2016, month: 1 });
console.log(grid.headers); // ['·ä•·àë·ãµ', ...]
console.log(grid.days); // Array of day objects
```

## Holiday Filtering

All holiday functions support filtering by tags:

```js
import { getHolidaysForYear, HolidayTags } from 'kenat';
const publicHolidays = getHolidaysForYear(2016, { filter: [HolidayTags.PUBLIC] });
```

## Geez Numeral Conversion

- `toGeez(number)` ‚Äî Converts numbers to Geez numerals.
- `toArabic(geezStr)` ‚Äî Converts Geez numerals to Arabic numbers.

## Date Validation & Utilities

- `isValidEthiopianDate(year, month, day)` ‚Äî Checks if a date is valid.
- `isEthiopianLeapYear(year)` ‚Äî Checks for leap years.
- `getWeekday({ year, month, day })` ‚Äî Returns the weekday index.

## Error Classes

Kenat exports error classes for robust error handling:

- `InvalidEthiopianDateError`
- `InvalidGregorianDateError`
- `InvalidInputTypeError`
- `InvalidTimeError`
- `InvalidGridConfigError`
- `UnknownHolidayError`
