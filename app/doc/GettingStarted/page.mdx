# üöÄ Getting Started

Kenat helps you create and manipulate Ethiopian dates using the intuitive `Kenat` class. It supports creating dates from various sources and lets you easily switch between Ethiopian and Gregorian formats.

---

## üèóÔ∏è Creating a New Date Instance

The `Kenat` constructor is flexible. Here are the ways you can create a new date.

### From the Current Date

If you create a `Kenat` instance with no arguments, it will automatically represent the current system date.

<Tabs
  items={[
    <div key="js" style={{ display: 'flex', alignItems: 'center', gap: '0.4rem' }}><SiJavascript size={16} /> JavaScript</div>,
    <div key="python" style={{ display: 'flex', alignItems: 'center', gap: '0.4rem' }}><SiPython size={16} /> Python</div>
  ]}
>
  <Tabs.Tab>
    ```js
    import Kenat from 'kenat';

    const today = new Kenat(); 

    console.log('ET:', today.getEthiopian());
    // Example Output ET: { year: 2017, month: 10, day: 5 }

    console.log('GC:', today.getGregorian());
    // Example Output GC: { year: 2025, month: 6, 13 }
    ```
  </Tabs.Tab>
  <Tabs.Tab>
    ```python
    from kenat import Kenat

    today = Kenat()

    print('ET:', today.get_ethiopian())
    # Example Output ET: { 'year': 2017, 'month': 10, 'day': 5 }

    print('GC:', today.get_gregorian())
    # Example Output GC: { 'year': 2025, 'month': 6, 'day': 13 }
    ```
  </Tabs.Tab>
</Tabs>

### From a JavaScript `Date` Object

You can pass a native JavaScript `Date` object directly to the constructor.

<Tabs
  items={[
    <div key="js" style={{ display: 'flex', alignItems: 'center', gap: '0.4rem' }}><SiJavascript size={16} /> JavaScript</div>,
    <div key="python" style={{ display: 'flex', alignItems: 'center', gap: '0.4rem' }}><SiPython size={16} /> Python</div>
  ]}
>
  <Tabs.Tab>
    ```js
    const gcDate = new Date('2024-08-22');
    const fromDate = new Kenat(gcDate);

    console.log(fromDate.getEthiopian());
    // Output: { year: 2016, month: 12, day: 13 }
    ```
  </Tabs.Tab>
  <Tabs.Tab>
    ```python
    from datetime import datetime
    from kenat import Kenat

    gc_date = datetime(2024, 8, 22)
    from_date = Kenat(gc_date)

    print(from_date.get_ethiopian())
    # Output: { 'year': 2016, 'month': 12, 'day': 13 }
    ```
  </Tabs.Tab>
</Tabs>

## üéâ Working with Holidays

Kenat also provides functions to get holidays for a given year.

<Tabs
  items={[
    <div key="js" style={{ display: 'flex', alignItems: 'center', gap: '0.4rem' }}><SiJavascript size={16} /> JavaScript</div>,
    <div key="python" style={{ display: 'flex', alignItems: 'center', gap: '0.4rem' }}><SiPython size={16} /> Python</div>
  ]}
>
  <Tabs.Tab>
    ```js
    import { getHolidaysForYear, HolidayTags } from 'kenat';

    const year = 2017; // E.C.

    // Get only public holidays by passing a single tag.
    const publicHolidays = getHolidaysForYear(year, {
        filter: HolidayTags.PUBLIC
    });

    console.log(`Found ${publicHolidays.length} public holidays in ${year} E.C.`);
    // Expected: Lists all holidays tagged as 'public'.
    ```
  </Tabs.Tab>
  <Tabs.Tab>
    ```python
    from kenat import get_holidays_for_year, HolidayTags

    year = 2017 # E.C.

    # Get only public holidays by passing a single tag.
    public_holidays = get_holidays_for_year(year, filter_by=HolidayTags.PUBLIC)

    print(f"Found {len(public_holidays)} public holidays in {year} E.C.")
    # Expected: Lists all holidays tagged as 'public'.
    ```
  </Tabs.Tab>
</Tabs>
