Kenat makes it simple to work with Ethiopian calendar dates in JavaScript.  
You can create a date instance, convert between calendars, and format dates easily.

## Basic usage with today's date

```js filename="app.js" showLineNumbers {3}
import Kenat, { gregorianToEthiopian, ethiopianToGregorian } from 'kenat';

const date = new Kenat(); // defaults to today
console.log(date);
console.log(date.getEthiopian());
console.log(date.getGregorian());
````

**What the imports do:**

* `Kenat`: A class that wraps Ethiopian date logic and provides useful methods.
* `gregorianToEthiopian(year, month, day)`: Converts a Gregorian date to an Ethiopian one.
* `ethiopianToGregorian(year, month, day)`: Converts an Ethiopian date to a Gregorian one.

---

## Creating a Kenat instance

If no date string is passed, `new Kenat()` defaults to the current system date (in Gregorian) and converts it to the Ethiopian calendar internally.

```js
const date = new Kenat();
```

Internally, it stores both the Ethiopian and Gregorian representations of the date.

---

## Viewing the output

Logging the Kenat instance will show its internal state:

```js
Kenat {
  ethiopian: { year: 2017, month: 9, day: 20 }
}
```

To get the date as an object:

```js
console.log(date.getEthiopian()); // { year: 2017, month: 9, day: 20 }
console.log(date.getGregorian()); // { year: 2025, month: 5, day: 28 }
```

---

## Using a custom Ethiopian date

You can also initialize the class with a specific Ethiopian date in `'yyyy/mm/dd'` format:

```js
const custom = new Kenat('2016/6/10');
console.log(custom.getGregorian()); // { year: 2024, month: 2, day: 18 }
```
