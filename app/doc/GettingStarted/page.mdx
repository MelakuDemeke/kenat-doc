# <FaRocket className="inline-block mr-2" /> Getting Started

Kenat helps you create and manipulate Ethiopian dates using the intuitive `Kenat` class. It supports creating dates from various sources and lets you easily switch between Ethiopian and Gregorian formats.

---

## <FaHammer className="inline-block mr-2" /> Creating a New Date Instance

The `Kenat` constructor is flexible. Here are the ways you can create a new date.

### Supported Input Types & Error Handling

Kenat automatically detects the type of input you provide:

- **No arguments**: Uses the current system date.
- **String**: Accepts `YYYY/MM/DD` or `YYYY-MM-DD` format (Ethiopian or Gregorian).
- **Date object**: Accepts native JavaScript `Date` or Python `datetime` objects.
- **Numbers**: Pass year, month, and day as separate numbers.

If you provide an invalid date (e.g., out-of-range month/day, wrong format), Kenat will throw an error describing the issue.

### Time Support

You can optionally pass time information (hour, minute, period) when creating a date:

  <Tabs.Tab>
    ```js
    // Use 'day' or 'night' for the period property
    const withTime = new Kenat(2017, 10, 5, { hour: 3, minute: 30, period: 'day' });
    // or
    const withTimeNight = new Kenat(2017, 10, 5, { hour: 3, minute: 30, period: 'night' });
    ```
  </Tabs.Tab>
  <Tabs.Tab>
    ```dart
    // Dart doesn't take year/month/day as separate constructor args — pass a
    // Time object as the second argument to fromEthiopian() instead.
    // Use 'day' or 'night' for the period property.
    final withTime = Kenat.fromEthiopian(2017, 10, 5, Time(3, 30, 'day'));
    // or
    final withTimeNight = Kenat.fromEthiopian(2017, 10, 5, Time(3, 30, 'night'));

    print(withTime.formatWithTime());      // ሰኔ 5 2017 03:30 ጠዋት
    print(withTimeNight.formatWithTime()); // ሰኔ 5 2017 03:30 ማታ
    ```
  </Tabs.Tab>
  <Tabs.Tab>
    ```python
    # Use 'day' or 'night' for the period property
    with_time = Kenat(2017, 10, 5, hour=3, minute=30, period='day')
    # or
    with_time_night = Kenat(2017, 10, 5, hour=3, minute=30, period='night')
    ```
  </Tabs.Tab>
  <Tabs.Tab>
    ```php
    use Kenat\Kenat;

    // PHP's constructor takes an array/EthiopianDate/string, not separate
    // y/m/d args — pass the date array and a time array together.
    // Use 'day' or 'night' for the period property.
    $withTime = new Kenat(
        ['year' => 2017, 'month' => 10, 'day' => 5],
        ['hour' => 3, 'minute' => 30, 'period' => 'day']
    );
    // or
    $withTimeNight = new Kenat(
        ['year' => 2017, 'month' => 10, 'day' => 5],
        ['hour' => 3, 'minute' => 30, 'period' => 'night']
    );

    echo $withTime->format(['includeTime' => true]);      // ሰኔ 5 2017 03:30 ጠዋት
    echo $withTimeNight->format(['includeTime' => true]); // ሰኔ 5 2017 03:30 ማታ
    ```
  </Tabs.Tab>
</Tabs>

### Output Types

Date methods like `getEthiopian()` and `getGregorian()` return plain objects, not Kenat instances.

### From the Current Date

If you create a `Kenat` instance with no arguments, it will automatically represent the current system date.

    ```dart
      // Example Output ET: {year: 2018, month: 11, day: 4}

      // Example Output GC: {year: 2026, month: 7, day: 11}
    ```

    ```python
    print('GC:', today.get_gregorian())
    # Example Output GC: { 'year': 2025, 'month': 6, 'day': 13 }
    ```
  </Tabs.Tab>
  <Tabs.Tab>
    ```php

    $today = new Kenat();

    // getEthiopian()/getGregorian() return typed EthiopianDate/GregorianDate
    // value objects, not arrays — access fields directly.
    echo "ET: {$today->getEthiopian()->year}-{$today->getEthiopian()->month}-{$today->getEthiopian()->day}";
    // Example Output ET: 2018-11-4

    $gc = $today->getGregorian();
    echo "GC: {$gc->year}-{$gc->month}-{$gc->day}";
    // Example Output GC: 2026-7-12
    ```

### From a Native Date Object

You can pass a native date object directly to the constructor: a JavaScript
`Date`, a Python `datetime`, or a Dart `DateTime`.

    ```python
    # Output: { 'year': 2016, 'month': 12, 'day': 13 }
    ```

    ```php
    $fromDate = new Kenat($gcDate);

    $et = $fromDate->getEthiopian();
    ```

## <FaGift className="inline-block mr-2" /> Working with Holidays

Kenat also provides functions to get holidays for a given year.

  <Tabs.Tab>
    ```js
    import { getHolidaysForYear, HolidayTags } from 'kenat';

    const year = 2017; // E.C.

    // Get only public holidays by passing a single tag.
    const publicHolidays = getHolidaysForYear(year, {
        filter: HolidayTags.PUBLIC
    });

    console.log(`Found ${publicHolidays.length} public holidays in ${year} E.C.`);
    // Expected: Lists all holidays tagged as 'public'.
    ```
  </Tabs.Tab>
  <Tabs.Tab>
    ```dart
    import 'package:kenat/kenat.dart';

    void main() {
      final year = 2017; // E.C.

      // Get only public holidays by passing a single tag.
      final publicHolidays = getHolidaysForYear(year, {
        'filter': HolidayTags.public,
      });

      print('Found ${publicHolidays.length} public holidays in $year E.C.');
      // Found 15 public holidays in 2017 E.C.
    }
    ```
  </Tabs.Tab>
  <Tabs.Tab>
    ```python
    from kenat import get_holidays_for_year, HolidayTags

    year = 2017 # E.C.

    # Get only public holidays by passing a single tag.

    print(f"Found {len(public_holidays)} public holidays in {year} E.C.")
    # Expected: Lists all holidays tagged as 'public'.
    ```
  </Tabs.Tab>
  <Tabs.Tab>

    ```php
        'filter' => HolidayTags::PUBLIC,
    ]);

    echo "Found " . count($publicHolidays) . " public holidays in {$year} E.C.";
    // Found 15 public holidays in 2017 E.C.
    ```
