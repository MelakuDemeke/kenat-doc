* `Kenat`: A class that provides Ethiopian calendar utilities.
* `gregorianToEthiopian(date)`: A helper function to convert a Gregorian date to an Ethiopian one.
* `ethiopianToGregorian(date)`: A helper function to convert an Ethiopian date to a Gregorian one.

---

### ðŸ”¹ `const date = new Kenat();`

This creates a new instance of the `Kenat` class.

* If no date is provided, it uses the **current system date** (in Gregorian calendar) and converts it to the Ethiopian calendar internally.
* Under the hood, it stores both versions: Gregorian and Ethiopian.

---

This means the Ethiopian date equivalent of May 28, 2025 (Gregorian) is 20/09/2017 (E.C).
