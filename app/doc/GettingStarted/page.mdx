# 🚀 Getting Started

Kenat makes it simple to work with Ethiopian calendar dates in JavaScript.  
You can create a date instance, convert between calendars, do date arithmetic, and format dates using modern or traditional Ethiopian conventions.

---

## 📦 Installation

Install via npm:

```bash
npm install kenat
````

Or load in the browser using ESM:

```html
<script type="module">
  import Kenat from 'https://cdn.jsdelivr.net/npm/kenat/+esm';
</script>
```

---

## ✨ Quick Usage

```js filename="app.js" showLineNumbers {3}
import Kenat, { toEC, toGC } from 'kenat';

const date = new Kenat(); // defaults to today
console.log(date);                       // Internal state
console.log(date.getEthiopian());       // → { year: 2017, month: 9, day: 20 }
console.log(date.getGregorian());       // → { year: 2025, month: 5, day: 28 }
```

---

## 🔍 What the Imports Do

* `Kenat`: Class wrapper around Ethiopian calendar logic
* `toEC(gYear, gMonth, gDay)`: Convert a Gregorian date to an Ethiopian date
* `toGC(eYear, eMonth, eDay)`: Convert an Ethiopian date to a Gregorian date

---

## 🎯 Creating a Kenat Instance

By default, `new Kenat()` uses the current system date (Gregorian), and internally converts it to Ethiopian:

```js
const date = new Kenat(); // ← today’s Ethiopian equivalent
```

Or initialize it with a custom Ethiopian date:

```js
const custom = new Kenat('2016/6/10');
console.log(custom.getGregorian()); 
// → { year: 2024, month: 2, day: 18 }
```

---

## 🧾 Viewing the Output

```js
Kenat {
  ethiopian: { year: 2017, month: 9, day: 20 },
  time: { hour: 12, minute: 0, period: 'day' }
}
```

To access the values as objects:

```js
console.log(date.getEthiopian());
// → { year: 2017, month: 9, day: 20 }

console.log(date.getGregorian());
// → { year: 2025, month: 5, day: 28 }
```
