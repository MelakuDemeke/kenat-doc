# <FiClock className="inline-block mr-2" /> Ethiopian Time Converter

The Ethiopian time system runs in two 12-hour cycles, starting from 6 AM and 6 PM. This interactive tool helps you convert between standard 24-hour Gregorian time and Ethiopian time using the `Time` class from the `kenat` library.
