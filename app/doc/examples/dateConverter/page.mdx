## 🧠 How It Works

This converter demonstrates how to use `kenat`'s date conversion utilities to go between Ethiopian (EC) and Gregorian (GC) calendars.

### 🔁 Function Signatures

- **Convert GC to EC**  
  ```js
  import { toEC } from 'kenat'
  const eth = toEC(2024, 6, 7)
  // → { year: 2016, month: 10, day: 30 }
