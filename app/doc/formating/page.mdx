# Formatting Ethiopian Dates with Kenat

The `kenat.formatting` module provides a collection of functions to format `Kenat` date objects into various string representations. You can combine these with the `Time` class for time-aware formatting.

---

## Formatting Functions

Here are the primary functions available for formatting dates in Python.

### `format_standard(et_date, lang='amharic')`

Formats a date into a standard `"Month Day Year"` string.

<Tabs
  items={[
    <div key="javascript" style={{ display: 'flex', alignItems: 'center', gap: '0.4rem' }}><SiJavascript size={16} /> JavaScript</div>,
    <div key="python" style={{ display: 'flex', alignItems: 'center', gap: '0.4rem' }}><SiPython size={16} /> Python</div>
  ]}
>
<Tabs.Tab>
```javascript
import { Kenat } from 'kenat';

const today = new Kenat(2016, 1, 10);

// Default Amharic output
console.log(today.format('MMMM D YYYY'));
// Output: መስከረም 10 2016

// English output
console.log(today.format('MMMM D YYYY', { lang: 'en' }));
// Output: Meskerem 10 2016
```
</Tabs.Tab>
<Tabs.Tab>
```python
from kenat import Kenat
from kenat.formatting import format_standard

today = Kenat('2016/1/10')

# Default Amharic output
print(format_standard(today))
# Output: መስከረም 10 2016

# English output
print(format_standard(today, lang='english'))
# Output: Meskerem 10 2016
```
</Tabs.Tab>
</Tabs>

### `format_with_weekday(et_date, lang='amharic', use_geez=False)`

Includes the day of the week in the output.

<Tabs
  items={[
    <div key="javascript" style={{ display: 'flex', alignItems: 'center', gap: '0.4rem' }}><SiJavascript size={16} /> JavaScript</div>,
    <div key="python" style={{ display: 'flex', alignItems: 'center', gap: '0.4rem' }}><SiPython size={16} /> Python</div>
  ]}
>
<Tabs.Tab>
```javascript
import { Kenat } from 'kenat';

const today = new Kenat(2016, 1, 10);

// Weekday in Amharic
console.log(today.format('dddd, MMMM D YYYY'));
// Output: ማክሰኞ, መስከረም 10 2016

// Weekday with Geez numerals
console.log(today.format('dddd, MMMM D YYYY', { useGeez: true }));
// Output: ማክሰኞ, መስከረም ፲ ፳፻፲፮
```
</Tabs.Tab>
<Tabs.Tab>
```python
from kenat import Kenat
from kenat.formatting import format_with_weekday

today = Kenat('2016/1/10')

# Weekday in Amharic
print(format_with_weekday(today))
# Output: ማክሰኞ, መስከረም 10 2016

# Weekday with Geez numerals
print(format_with_weekday(today, use_geez=True))
# Output: ማክሰኞ, መስከረም ፲ ፳፻፲፮
```
</Tabs.Tab>
</Tabs>

### `format_with_time(et_date, time_obj, lang='amharic')`

Combines a date and a `Time` object into a single string.

<Tabs
  items={[
    <div key="javascript" style={{ display: 'flex', alignItems: 'center', gap: '0.4rem' }}><SiJavascript size={16} /> JavaScript</div>,
    <div key="python" style={{ display: 'flex', alignItems: 'center', gap: '0.4rem' }}><SiPython size={16} /> Python</div>
  ]}
>
<Tabs.Tab>
```javascript
import { Kenat } from 'kenat';

const today = new Kenat(2016, 1, 10, 8, 30);

console.log(today.format('MMMM D YYYY h:mm A'));
// Output: መስከረም 10 2016 8:30 ቀን
```
</Tabs.Tab>
<Tabs.Tab>
```python
from kenat import Kenat, Time
from kenat.formatting import format_with_time

today = Kenat('2016/1/10')
time = Time(8, 30, 'day')

print(format_with_time(today, time))
# Output: መስከረም 10 2016 8:30 ቀን
```
</Tabs.Tab>
</Tabs>

### `to_iso_date_string(et_date, time_obj=None)`

Returns an ISO-like string, with optional time.

<Tabs
  items={[
    <div key="javascript" style={{ display: 'flex', alignItems: 'center', gap: '0.4rem' }}><SiJavascript size={16} /> JavaScript</div>,
    <div key="python" style={{ display: 'flex', alignItems: 'center', gap: '0.4rem' }}><SiPython size={16} /> Python</div>
  ]}
>
<Tabs.Tab>
```javascript
import { Kenat } from 'kenat';

// Date only
const today = new Kenat(2016, 1, 10);
console.log(today.toISOString());
// Output: 2016-01-10

// With time
const withTime = new Kenat(2016, 1, 10, 8, 30);
console.log(withTime.toISOString());
// Output: 2016-01-10T08:30
```
</Tabs.Tab>
<Tabs.Tab>
```python
from kenat import Kenat, Time
from kenat.formatting import to_iso_date_string

today = Kenat('2016/1/10')

# Date only
print(to_iso_date_string(today))
# Output: 2016-01-10

# With time
time = Time(8, 30, 'day')
print(to_iso_date_string(today, time))
# Output: 2016-01-10T08:30
```
