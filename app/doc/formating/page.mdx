# Formatting Ethiopian Dates with Kenat


The `kenat.formatting` module provides a collection of functions to format `Kenat` date objects into various string representations. You can combine these with the `Time` class for time-aware formatting.

You can format dates in two ways:

- Using instance methods on a `Kenat` object (e.g., `date.format(...)`, `date.formatInGeezAmharic()`, `date.toISOString()`).
- Using named formatting functions (e.g., `formatStandard(date)`, `formatWithWeekday(date)`, etc.) imported from `kenat`.

All formatting functions return strings suitable for display. Invalid date or time objects will throw an error (e.g., `InvalidEthiopianDateError`, `InvalidTimeError`).

## Formatting Functions

Here are the primary functions available for formatting dates in JavaScript. All formatting functions can be used as instance methods on a `Kenat` object or as named functions imported from `kenat`. Invalid date or time objects will throw an error (e.g., `InvalidEthiopianDateError`, `InvalidTimeError`).


### Standard Date Format

Formats a date into a standard "Month Day Year" string.

<Tabs
  items={[
    <div key="javascript" style={{ display: 'flex', alignItems: 'center', gap: '0.4rem' }}><SiJavascript size={16} /> JavaScript</div>,
    <div key="dart" style={{ display: 'flex', alignItems: 'center', gap: '0.4rem' }}><SiDart size={16} /> Dart</div>,
    <div key="python" style={{ display: 'flex', alignItems: 'center', gap: '0.4rem' }}><SiPython size={16} /> Python</div>
  ]}
>
<Tabs.Tab>
```javascript
import { Kenat, formatStandard } from 'kenat';

// Kenat constructor supports string/object/Date
const today = new Kenat('2016/1/10');
// or: new Kenat({ year: 2016, month: 1, day: 10 })
// or: new Kenat(new Date(2016, 0, 10))

// Using Kenat instance method (default Amharic)
console.log(today.format());
// Output: መስከረም 10 2016

// Using Kenat instance method (English)
console.log(today.format({ lang: 'english' }));
// Output: Meskerem 10 2016

// Using named formatting function
console.log(formatStandard(today));
// Output: መስከረም 10 2016
console.log(formatStandard(today, 'english'));
// Output: Meskerem 10 2016
```
</Tabs.Tab>
<Tabs.Tab>
```dart
import 'package:kenat/kenat.dart';

// Kenat's plain constructor accepts a string, a map, or a DateTime
final today = Kenat('2016/1/10');
// or: Kenat({ 'year': 2016, 'month': 1, 'day': 10 })
// or: Kenat(DateTime(2016, 1, 10))

// Using Kenat instance method (default Amharic)
print(today.format());
// Output: መስከረም 10 2016

// Using Kenat instance method (English)
print(today.format(lang: 'english'));
// Output: Meskerem 10 2016

// Using the named formatting function — note it takes the raw
// {year, month, day} map, not the Kenat instance itself
print(formatStandard(today.getEthiopian()));
// Output: መስከረም 10 2016
print(formatStandard(today.getEthiopian(), 'english'));
// Output: Meskerem 10 2016
```
</Tabs.Tab>
<Tabs.Tab>
```python
from kenat import Kenat
from kenat.formatting import format_standard

today = Kenat('2016/1/10')

# Default Amharic output
print(format_standard(today))
# Output: መስከረም 10 2016

# English output
print(format_standard(today, lang='english'))
# Output: Meskerem 10 2016
```

### Date Format with Weekday

Includes the day of the week in the output.

<Tabs.Tab>
```javascript
import { Kenat, formatWithWeekday } from 'kenat';

const today = new Kenat('2016/1/10');

// Using Kenat instance method
console.log(today.formatWithWeekday());
// Output: ማክሰኞ, መስከረም 10 2016
console.log(today.formatWithWeekday('english', true));
// Output: Tuesday, Meskerem ፲ ፳፻፲፮

// Using named formatting function
console.log(formatWithWeekday(today));
// Output: ማክሰኞ, መስከረም 10 2016
console.log(formatWithWeekday(today, 'english', true));
// Output: Tuesday, Meskerem ፲ ፳፻፲፮
// Options: lang ('amharic'|'english'), useGeez (boolean)
```
</Tabs.Tab>
<Tabs.Tab>
```dart

// Using Kenat instance method
print(today.formatWithWeekday());

print(today.formatWithWeekday('english', true));

// Using named formatting function (takes the {year, month, day} map)

// Output: ሐሙስ, መስከረም 10 2016
```

### Date Format with Time

Combines a date and a Time object into a single string.

<Tabs
  items={[
    <div key="javascript" style={{ display: 'flex', alignItems: 'center', gap: '0.4rem' }}><SiJavascript size={16} /> JavaScript</div>,
    <div key="dart" style={{ display: 'flex', alignItems: 'center', gap: '0.4rem' }}><SiDart size={16} /> Dart</div>,
    <div key="python" style={{ display: 'flex', alignItems: 'center', gap: '0.4rem' }}><SiPython size={16} /> Python</div>
  ]}
>
<Tabs.Tab>
```javascript
import { Kenat, Time, formatWithTime } from 'kenat';

const today = new Kenat('2016/1/10');
const time = new Time(8, 30, 'day');

// Using Kenat instance method
console.log(today.formatWithTime(time));
// Output: መስከረም 10 2016 8:30 ቀን

// Using named formatting function
console.log(formatWithTime(today, time));
// Output: መስከረም 10 2016 8:30 ቀን
// Options: lang ('amharic'|'english'), useGeez (boolean)
```
</Tabs.Tab>
<Tabs.Tab>
```dart
import 'package:kenat/kenat.dart';

final today = Kenat('2016/1/10');
final time = Time(8, 30, 'day');

// The Kenat instance method doesn't take a Time argument — attach the
// time to the instance first with setTime(), which returns a new Kenat.
final todayWithTime = today.setTime(time);
print(todayWithTime.formatWithTime());
// Output: መስከረም 10 2016 08:30 ጠዋት

// The named formatting function does take a Time directly
print(formatWithTime(today.getEthiopian(), time));
// Output: መስከረም 10 2016 08:30 ጠዋት
// Options: lang ('amharic'|'english')
```
</Tabs.Tab>
<Tabs.Tab>
```python
from kenat import Kenat, Time
from kenat.formatting import format_with_time

today = Kenat('2016/1/10')
time = Time(8, 30, 'day')

print(format_with_time(today, time))
# Output: መስከረም 10 2016 8:30 ቀን
```

### ISO Date String Format

Returns an ISO-like string, with optional time.

<Tabs
  items={[
    <div key="javascript" style={{ display: 'flex', alignItems: 'center', gap: '0.4rem' }}><SiJavascript size={16} /> JavaScript</div>,
    <div key="dart" style={{ display: 'flex', alignItems: 'center', gap: '0.4rem' }}><SiDart size={16} /> Dart</div>,
    <div key="python" style={{ display: 'flex', alignItems: 'center', gap: '0.4rem' }}><SiPython size={16} /> Python</div>
  ]}
>
<Tabs.Tab>
```javascript
import { Kenat, Time, toISODateString } from 'kenat';

// Date only
const today = new Kenat('2016/1/10');
console.log(today.toISOString());
// Output: 2016-01-10
console.log(toISODateString(today));
// Output: 2016-01-10

// With time
const time = new Time(8, 30, 'day');
// Kenat does not accept time in constructor, set time via setTime
today.setTime(8, 30, 'day');
console.log(today.toISOString());
// Output: 2016-01-10T08:30
console.log(toISODateString(today, time));
// Output: 2016-01-10T08:30
```
</Tabs.Tab>
<Tabs.Tab>
```dart
import 'package:kenat/kenat.dart';

final today = Kenat('2016/1/10');

// Note: a string-constructed Kenat always carries a default noon-day
// time internally, so toISOString() already includes a time component.
print(today.toISOString());
// Output: 2016-01-10T12:00

// The named function takes the raw date map with no implicit time:
print(toISODateString(today.getEthiopian()));
// Output: 2016-01-10

// With an explicit time — setTime() returns a NEW Kenat instance
final todayWithTime = today.setTime(8, 30, 'day');
print(todayWithTime.toISOString());
// Output: 2016-01-10T08:30
print(toISODateString(today.getEthiopian(), { 'hour': 8, 'minute': 30, 'period': 'day' }));
// Output: 2016-01-10T08:30
```
</Tabs.Tab>
<Tabs.Tab>
```python
from kenat import Kenat, Time
from kenat.formatting import to_iso_date_string

today = Kenat('2016/1/10')

# Date only
print(to_iso_date_string(today))
# Output: 2016-01-10

# With time
time = Time(8, 30, 'day')
print(to_iso_date_string(today, time))
# Output: 2016-01-10T08:30
```
