    ## ✅ Supported Environments

    - Node.js ≥ 14  
    - Modern browsers (ES6+)  
    - Works with frameworks like React, Vue (via wrappers), Svelte  
    - Not tested on IE11 or legacy browsers

    ## 📥 Import & Basic Usage

    ```js
    import Kenat from 'kenat';

    const date = new Kenat();
    console.log(date.getEthiopian()); // { year: 2017, month: 9, day: 15 }
    ```

    ## 🌀 Dynamic Import (CommonJS)

    If you're using `require()` or CommonJS modules:

    ```js
    (async () => {
      const { default: Kenat } = await import('kenat');
      console.log(new Kenat().toString());
    })();
    ```
    ## 🔍 Verify Installation

    Quick one-liner to check if it's working:

    ```bash
    node -e "import('kenat').then(m => console.log(new m.default().toString()))"
    ```
    Output
    ```bash
    Ethiopian: 2017-9-21
    ```
