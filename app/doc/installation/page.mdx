    ## <FiDownload className="inline-block mr-2" /> Import & Usage

    Kenat supports both default and named imports:

    ```js
    import Kenat, { getHolidaysForYear, HolidayTags, toEC, toGC, formatStandard } from 'kenat';

    // Create a date
    const date = new Kenat();
    console.log(date.getEthiopian());

    // Use named exports
    const holidays = getHolidaysForYear(2017, { filter: HolidayTags.PUBLIC });
    console.log(holidays);

    // Convert Gregorian to Ethiopian
    const etDate = toEC(2025, 7, 18); // { year, month, day }
    console.log(etDate);

    // Format an Ethiopian date
    const formatted = formatStandard({ year: 2017, month: 10, day: 5 }, 'english');
    console.log(formatted);
    ```

    ## <FiRefreshCw className="inline-block mr-2" /> ES Modules & CommonJS

    Kenat is published as an ES module. For CommonJS, use dynamic import:

    ```js
    (async () => {
      const { default: Kenat, getHolidaysForYear } = await import('kenat');
      console.log(new Kenat().toString());
      console.log(getHolidaysForYear(2017));
    })();
    ```

    ## <FiDownload className="inline-block mr-2" /> Import & Usage

    ```dart
    import 'package:kenat/kenat.dart';

    void main() {
      // Create a date
      final date = Kenat.now();
      print(date.getEthiopian());

      // Use named exports
      final holidays = getHolidaysForYear(2017, { 'filter': HolidayTags.public });
      print(holidays.length);

      // Convert Gregorian to Ethiopian
      final etDate = toEC(2025, 7, 18); // {year, month, day}
      print(etDate);

      // Format an Ethiopian date
      final formatted = formatStandard({ 'year': 2017, 'month': 10, 'day': 5 }, 'english');
      print(formatted);
    }
    ```

    ## <FiCheckCircle className="inline-block mr-2" /> Supported Environments

    - PHP ≥ 8.1
    - PSR-4 autoloading, zero runtime dependencies (requires only `ext-json`)
    - Immutable, typed value objects — works great with static analysis (PHPStan/Psalm)
    - Published on [Packagist](https://packagist.org/packages/melakudemeke/kenat-php)

    ## <FiDownload className="inline-block mr-2" /> Import & Usage

    ```php
    use Kenat\Kenat;
    use Kenat\Holidays;
    use Kenat\HolidayTags;
    use Kenat\Conversions;

    // Create a date
    $date = new Kenat();
    echo $date->format(); // e.g. "ጥቅምት 5 2018"

    // Get holidays
    $holidays = Holidays::getHolidaysForYear(2017, ['filter' => HolidayTags::PUBLIC]);
    echo count($holidays);

    // Convert Gregorian to Ethiopian
    $etDate = Conversions::toEC(2025, 7, 18); // EthiopianDate{ year, month, day }
    echo $etDate->year;

    // Format an Ethiopian date
    $formatted = (new Kenat('2017/10/5'))->format(['lang' => 'english']);
    echo $formatted;
    ```

    ## <FiFolder className="inline-block mr-2" /> Quick Reference: Common Classes

    - `Kenat` — the main date class (immutable; every mutating method returns a *new* instance)
    - `Time` — Ethiopian 12-hour day/night time class
    - `Conversions::toEC()`, `Conversions::toGC()`
    - `GeezConverter::toGeez()`, `GeezConverter::toArabic()`
    - `Holidays::getHolidaysForYear()`, `Holidays::getHolidaysInMonth()`, `Holidays::getHoliday()`, `HolidayTags`
    - `BahireHasab::calculate()`
    - `Fasting::getFastingPeriod()`, `FastingKeys`
    - `MonthGrid::create()`
    - `Formatting::formatStandard()`, `formatWithWeekday()`, `formatWithTime()`, `formatInGeezAmharic()`, `formatShort()`, `toISODateString()`
    - `DayArithmetic::addDays()`, `addMonths()`, `addYears()`, `diffInDays()`, `diffInMonths()`, `diffInYears()`

    ## <FiAlertTriangle className="inline-block mr-2" /> Error Handling

    Invalid input (e.g., an out-of-range date, a malformed string) throws a typed exception from `Kenat\Exceptions`, such as `InvalidEthiopianDateException` or `InvalidDateFormatException`.
